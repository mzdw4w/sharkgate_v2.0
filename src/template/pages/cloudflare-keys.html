Frequently Asked Questions

Where do I find my Cloudflare API key?

Login to your Cloudflare account for parentsoutloud.com

  • Go to My Profile.
  • Scroll down to API Keys and locate Global API Key.
  • Click the View button on the API Key to see your API identifier

See a cloudflare article here explaining this same process
